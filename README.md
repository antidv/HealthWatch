# HealthWatch

El proyecto propone un sistema basado en IoT para el monitoreo continuo de adultos mayores
en centros especializados. El objetivo es mejorar la calidad de vida y optimizar la respuesta
médica mediante tecnología proactiva y centrada en el ser humano.
